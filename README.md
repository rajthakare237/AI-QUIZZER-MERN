## üöÄ Key Features

### ü§ñ AI-Powered Quiz Generation
* **Instant Creation:** Simply enter a topic, and our AI generates relevant questions instantly.
* **Customizable:** Select the number of questions and choose your difficulty level (Easy, Medium, Hard).

### üîê User Authentication
* **Secure Access:** Full login and signup functionality using JWT (JSON Web Tokens).
* **Route Protection:** Middleware ensures only authenticated users can access quizzes and history.

### ‚è±Ô∏è Interactive Quiz Player
* **Real-time Timer:** Keeps users engaged with a countdown for each quiz.
* **Smooth UX:** Seamless transition between multiple-choice questions without page reloads.

### üìä Results & Analytics
* **Instant Grading:** Scores are calculated immediately upon submission.
* **Performance Review:** Detailed breakdown of correct vs. incorrect answers and percentage score.

### üìú History & Management
* **Attempt History:** Users can view all past quizzes to track improvement over time.
* **Profile Management:** Update avatars and user details easily.
* **Shareable Quizzes:** Generate unique links to challenge friends on specific topics.
